# Zeron Engine
A component based C++ Pong game built on a custom renderer as a Vancouver Film School Project.

#### Dependencies:
- CMake 3.8+, https://cmake.org/download/

#### How to Install:
- Run 'GenerateProject.bat' to create project solution
